# Backend Stage 1 API

## Base URL
https://your-domain.com

## Endpoints

### POST /api/profiles
Create profile

### GET /api/profiles/:id
Get single profile

### GET /api/profiles
Filter:
- gender
- country_id
- age_group

### DELETE /api/profiles/:id
Delete profile

## Tech Stack
- Node.js
- Express
- MongoDB

## Features
- Idempotent profile creation
- External API integration
- Filtering
- Proper error handling

## Deployment
Hosted on Railway
